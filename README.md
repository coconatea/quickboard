# 桌面剪贴板 QuickBoard

把常用的账号密码、文本、链接、代码片段集中存放，**随时一键复制、快速填写各类资料**。

- 🔒 **隐私安全**：数据**全部存在你自己的电脑上，不上传任何服务器**，账号密码、证件号等敏感信息完全不用担心泄露。
- ⚡ **下载即用**：不用安装、不用看代码，下载压缩包解压后双击就能用。
- 💾 **关了还在**：本地持久化保存，关闭程序、重启电脑，内容都还在。
- ⛄ **可爱不挡事**：窗口还能缩成一只 3D 雪人桌面宠物，悬停会俏皮一缩。

## ✨ 功能一览

- 🔍 **实时搜索**：顶部输入框按"名称 / 内容"即时过滤，`Ctrl + Shift + V` 快速聚焦搜索框。
- 🗂 **分类管理**：全部 / 账号密码 / 文本 / 链接 / 代码，可和搜索组合使用。
- 📋 **一键复制**：点条目上的复制按钮，内容自动写入系统剪贴板，直接去别处粘贴。
- ➕ **新建 / ✏️ 编辑 / 🗑 删除**：右下「+ 新建」添加；条目悬停出现编辑 / 删除（删除有二次确认，防误删）。
- 🌗 **深 / 浅主题**：右下太阳 / 月亮图标切换，选择会被记住。
- 📤 **导出备份**：右下导出按钮，把全部内容导出成 `snippets-时间戳.json` 到"下载"文件夹（不覆盖旧文件）。
- ⛄ **缩小成雪人宠物**：点窗口右上「—」，整个窗口缩成一只 3D 雪人；
  - 鼠标悬停：雪人俏皮一缩；
  - 单击雪人：恢复成完整窗口；
  - 按住雪人拖动：把它放到屏幕任意位置（系统级平滑，不抖动）。
- 💾 **本地持久化**：所有改动即时存盘，关闭后再打开内容原样保留。

## 📖 详细使用说明

### 1. 添加一条资料
点击右下角 **「+ 新建」**，填写：
- **名称**：方便你搜到的标签，例如"工商银行""家里 WiFi"；
- **内容**：真正要复制的文本，如卡号、密码、地址；
- **分类**：选"账号密码 / 文本 / 链接 / 代码"之一，方便归类。
保存后它会出现在列表里，并立刻写入本地存储。

### 2. 快速找到并复制
- 在顶部搜索框输入关键字（名称或内容都算），列表实时过滤；
- 找到后点条目右侧的 **复制按钮**，内容进入剪贴板，去网页 / 软件里直接 `Ctrl/Cmd + V` 粘贴即可——再也不用手忙脚乱翻备忘录。

### 3. 编辑与删除
- **编辑**：条目悬停时出现 ✏️，点它修改内容后保存；
- **删除**：条目悬停时出现 🗑，点它会有一次"确定要删除吗"的确认，避免误删。

### 4. 切换主题
点右下角太阳 / 月亮图标，在深色与浅色之间切换，你的选择会被记住，下次打开还是这个主题。

### 5. 导出与备份
点右下角导出按钮，会把当前所有资料导出为一个 `snippets-时间戳.json` 文件到你的"下载"文件夹。换电脑或想留个备份时，把这个文件存好即可（当前版本导入功能可后续添加）。

### 6. 把窗口变成雪人宠物
点窗口右上角的 **「—」**，整个窗口会缩成 170×170 的 3D 雪人，常驻桌面角落：
- **悬停**它：雪人俏皮地缩一下；
- **单击**它：立刻恢复成完整窗口；
- **按住拖动**它：把它移到屏幕任意位置，拖动过程系统级平滑、不抖动。

### 7. 数据存在哪里（如何查看 / 备份）
所有数据**只在本机**，不上传网络：
- **Windows**：`C:\Users\<你>\AppData\Roaming\desktop-clipboard\QuickBoard\snippets.json`
- **macOS**：`~/Library/Application Support/QuickBoard/snippets.json`

你可以直接用记事本 / 文本编辑器打开这个 `snippets.json` 查看自己的内容；想整体备份或换电脑，把这个文件拷走即可。首次启动会自动写入几条示例，清空后也会保留空列表。
